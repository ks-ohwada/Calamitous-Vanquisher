ABOUT "CALAMITOUS VANQUISHER"

Calamitous Vanquisher のリニューアルサイトです。

当サイトは2006年に発売されたDS用のソフト、"メトロイドプライムハンターズ" のコミュニティサイトとして活動しておりましたが、 ゲームのWi-Fiサービス終了に伴い閉鎖。しかしながら10年の時を経て、リニューアルオープンいたしました。

メトロイドファンに向けたコンテンツを載せていきますので、どうぞお楽しみください。
